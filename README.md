# GovCh 项目

这是一个基于 Python 的 Web 应用项目。

## 目录结构

```
GovCh/
├── app/              # 主应用代码目录
├── migrations/       # 数据库迁移文件
├── tests/            # 测试代码
├── static/           # 静态资源文件（CSS, JavaScript, 图片等）
├── templates/        # HTML 模板文件
├── docs/             # 项目文档
├── requirements/     # 依赖包配置文件
├── tools/            # 工具脚本
├── .env/             # 环境变量配置
└── README.md         # 项目说明文件
```

## 安装与运行

1. 克隆项目到本地
2. 安装依赖
3. 配置环境变量
4. 运行应用

## 开发说明

- 使用 Python 3.7+
- 遵循 PEP 8 代码规范
- 编写单元测试
